CSE 470 HW#2: Rotating Cubes

Author: Jacob Shaeffer

Date: February 15, 2017

Description: This program allows users to add rotating cubes to the canvas

Functionality: The four buttons will change the axis of rotation of the cubes to the
axis described on the button. The slider beneith will change the size of the cubes.

Parameters: scaleLoc is the array used to pass the scale and translation data to the gpu
scale is the value used to store the current size of the cubes
translate is a 2d array holding the x,y,z translation of each cube
numCubes holds the number of cubes that are being rendered

Resources:Prof. Angel's codes, as well as help from WebGL Fundamentals
